Spinner Test Page

This page demonstrates a working spinner implementation. Click the buttons below to test the spinner functionality.

Spinner Implementation Code

// HTML
<div class="spinner-container" id="spinner">
    <div class="spinner"></div>
    <div class="spinner-text">Loading...</div>
</div>

// CSS (see style section above)

// JavaScript
function showSpinner() {
    document.getElementById('spinner').classList.add('active');
}

function hideSpinner() {
    document.getElementById('spinner').classList.remove('active');
}

// Event listeners
document.getElementById('showSpinner').addEventListener('click', showSpinner);
document.getElementById('hideSpinner').addEventListener('click', hideSpinner);
